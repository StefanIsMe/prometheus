## Prerequisites

- Docker (running)
- An LLM API key from any [supported provider](/llm-providers/overview) (OpenAI, Anthropic, Google, etc.)

## Installation

<Tabs>
  <Tab title="curl">
    ```bash
    git clone https://github.com/StefanIsMe/prometheus.git && cd prometheus && ./scripts/install.sh
    ```
  </Tab>
  <Tab title="pipx">
    ```bash
    pipx install prometheus-agent
    ```
  </Tab>
</Tabs>

## Configuration

Set your LLM provider:

```bash
export PROMETHEUS_LLM="openai/gpt-5.4"
export LLM_API_KEY="your-api-key"
```

<Tip>
For best results, use `openai/gpt-5.4`, `anthropic/claude-opus-4-6`, or `openai/gpt-5.2`.
</Tip>

## Run Your First Scan

```bash
prometheus --target ./your-app
```

<Note>
First run pulls the Docker sandbox image automatically. Results are saved to `prometheus_runs/<run-name>`.
</Note>

## Target Types

Prometheus accepts multiple target types:

```bash
# Local codebase
prometheus --target ./app-directory

# GitHub repository
prometheus --target https://github.com/org/repo

# Live web application
prometheus --target https://your-app.com

# Multiple targets (white-box testing)
prometheus -t https://github.com/org/repo -t https://your-app.com
```
